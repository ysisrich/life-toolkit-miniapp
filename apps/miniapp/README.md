# 生活工具箱 (Life Toolkit Mini-Program)

一款极致追求“高颜值”与“极度丝滑体验”的个人生活工具箱小程序。

采用深色模式（Dark Mode）、苹果级毛玻璃质感（Glassmorphism）、以及顺滑的 GSAP 物理动画，让每一次日常记录都成为一种享受。

## ✨ 核心特性

- **顶级视觉美学**：全局深色星空渐变背景搭配半透明磨砂玻璃面板，彻底摒弃传统扁平化，引入充满高级感的 3D 图标。
- **动态交互架构**：整合业界顶尖的 GSAP 动画引擎，每一次页面进入、卡片点击，都有媲美 iOS 原生体验的阻尼回弹和错位弹射入场动画。
- **系统级推送能力**：内置配置中心，可通过接入 Bark 服务，将“喝水”、“吃药”等重要提醒直接推送到你的 iOS 设备通知中心。
- **渐进式功能矩阵**：
  - ✂️ **剪指甲** (已实装): 高颜值指甲修剪记录。
  - 💧 **喝水记录** (开发中)
  - 🌙 **睡眠监测** (规划中)
  - 💊 **吃药提醒** (规划中)

## 🛠️ 技术栈

- 微信小程序原生框架 (WXML / WXSS / JS)
- `npm` 模块化工程支持
- `gsap` - 负责所有丝滑的物理与关键帧动效处理
- 原生深色模式与 Skyline 渲染引擎适配

## 🚀 快速开始

1. **拉取代码**：
   将本项目代码克隆或下载到本地。
2. **导入项目**：
   打开 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)，选择“导入项目”，并选中本项目的根目录。
3. **安装依赖**：
   在小程序根目录打开终端，运行命令：
   ```bash
   npm install
   ```
4. **构建 npm**：
   在微信开发者工具顶部菜单栏中，点击 **工具 -> 构建 npm**。
5. **开启体验**：
   在开发者工具中编译预览即可。如果预览界面没有显示动画，请确保“详情 -> 本地设置”中勾选了**使用 npm 模块**。

## ⚙️ Bark 通知配置说明

为了让小工具拥有系统推送的能力，你可以在小程序的“设置”页面配置 Bark 密钥。
1. 在 iPhone App Store 搜索并下载 `Bark`。
2. 打开 Bark 获取你的专属 Key。
3. 点击小程序主页右上角的“齿轮”图标，将 Key 填入即可实现随时随地的测试和提醒推送。
